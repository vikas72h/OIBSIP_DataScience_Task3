# OIBSIP_DataScience_Task3
Car Price Prediction with Machine Learning: Developed a Machine Learning model to predict car selling prices based on various features such as present price, car age, fuel type, transmission, and kilometers driven. The project involved data preprocessing, visualization, model training, and performance evaluation. 🚗📊

Objective: Build a Machine Learning model to accurately predict car selling prices using historical car data and vehicle features.

Tools & Technologies:

1. Python
2. Pandas
3. NumPy
4. Matplotlib
5. Seaborn
6. Scikit-learn

Steps Performed:

#Loaded and cleaned the car dataset.
#Performed exploratory data analysis (EDA).
#Created a new feature (Car Age) from the manufacturing year.
#Encoded categorical variables for machine learning.
#Visualized data distribution and feature correlations.
#Trained a Random Forest Regression model.
#Predicted car selling prices using the trained model.
#Evaluated model performance using MAE, RMSE, and R² Score.

Outcome: Successfully developed a Machine Learning model that predicts car selling prices with good accuracy and identified the key factors influencing vehicle resale value.
